# Self-study-Map
神様はいつも私たちのそばにいます。


だから、自分が歩んできた人生を死後に振り返ってみると、必ず別の一行の足跡が見えてくる。


人生は、いつも身近な力に支えられている。 


でもね、一番苦しくて、悲しい時には、足跡は一行しか残らないのよ。


その時、神は立ち止まっている人の足となる。 


行き詰まると思っていた道も、きっと先に延びていく。




神一直在我们身边。


所以，在死后回顾自己走过的人生道路时，一定会看到另一行足迹。


人生呢，一直是被身边的力量支持着的。


但是呢，在最痛苦、最悲伤的时候，足迹就会只剩一行了。


那时候，神会成为这个停滞不前的人的双脚。


就连原本以为会停滞不前的道路，也一定会向前延伸的。
